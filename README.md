# FraudDetect-AI

An AI-powered fraud detection dashboard with a financial theme, built with **MiMo V2.5**.

## Features

- Real-time transaction monitoring
- Risk scoring with CSS heatmap visualization
- Alert cards for suspicious activities
- Transaction table with filtering
- Fraud pattern detection simulation
- Financial metrics dashboard
- Geographic risk mapping

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- CSS Grid heatmap visualization
- Simulated ML fraud scoring
- Zero dependencies

## Getting Started

1. Open `index.html` in a modern web browser
2. View the transaction feed with live risk scores
3. Click on high-risk transactions for detailed alerts
4. Monitor fraud patterns on the heatmap

## Risk Levels

- **Critical** (red): Score > 0.85 — immediate action required
- **High** (orange): Score 0.6–0.85 — manual review needed
- **Medium** (yellow): Score 0.3–0.6 — monitoring recommended
- **Low** (green): Score < 0.3 — normal activity

Built with MiMo V2.5 AI assistance.
